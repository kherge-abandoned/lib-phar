Usage
-----

Requirement
-----------

- PHP >= 5.3.3
- [Phine Exception] >= 1.0.0

Installation
------------

Via [Composer][]:

    $ composer require "phine/phar=~1.0"

Documentation
-------------

You can find the documentation in the [`docs/`](docs/) directory.
