Phine\Phar\File\Reader::isEndOfFile ( )

Checks if the end of the file has been reached.

Before feof() is called, an attempt is made to reach a single byte from the file. If the read fails, then the feof() is made to actually determine the if the end of the file has been reached. If the EOF is not reached, the file handle will be seeked back the one byte that was read.

Returns
boolean If the end of the file has been reached, false is returned. If the end of the file has not been reached, true is returned.
